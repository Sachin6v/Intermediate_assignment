# Restaurant Sales Analysis using Python

## Project Overview

This project analyzes restaurant billing data using Python. The goal is to understand customer behavior, sales trends, and tipping patterns through data analysis and visualization.

## Dataset

The dataset used is the built-in **tips** dataset from the **Seaborn** library.

It contains the following columns:

* total_bill
* tip
* sex
* smoker
* day
* time
* size

## Tools & Libraries Used

* Python
* Pandas
* Matplotlib
* Seaborn

## Key Performance Indicators (KPIs)

* Total Revenue
* Average Bill Amount
* Average Tip Amount
* Highest Bill
* Tip Percentage

## Analysis Performed

* Day-wise Revenue Analysis
* Lunch vs Dinner Revenue Comparison
* Smoker vs Non-Smoker Tip Analysis
* Top 5 Highest Bills
* Tip Percentage Calculation
* Bill vs Tip Relationship

## Visualizations

* Bar Charts
* Pie Chart
* Histogram
* Scatter Plot

## Key Insights

* Dinner generated more revenue than Lunch.
* Weekend days had higher customer spending.
* Higher bills usually resulted in higher tips.
* Tip behavior varied across customer groups.

## Conclusion

This project demonstrates how Python can be used for business data analysis and extracting useful insights from raw data.
