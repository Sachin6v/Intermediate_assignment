# Restaurant Sales Analysis

## Project Overview
This project analyzes restaurant sales data using the Tips dataset using Python.  
The objective is to understand customer spending behavior, tipping patterns, and factors affecting total bill amounts.

## Dataset Used
Tips Dataset (Seaborn Built-in Dataset)

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed
- Data Cleaning and Basic Exploration
- Average Total Bill Analysis
- Tip Distribution Analysis
- Gender-wise Spending Comparison
- Time-wise Sales Analysis (Lunch / Dinner)
- Day-wise Revenue Trends
- Relationship between Total Bill and Tip
- Customer Size Impact on Bill Amount

## Key Insights
- Dinner time generated higher bill amounts than lunch.
- Larger groups generally had higher total bills.
- Tips increased with total bill amount.
- Weekend days showed stronger customer spending.

## Conclusion
This project demonstrates how Python can be used for Exploratory Data Analysis (EDA) to generate business insights from restaurant sales data.
