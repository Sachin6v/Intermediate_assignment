# Shark Tank India Analysis

## Project Overview
This project analyzes Shark Tank India dataset using Python.  
The objective is to understand startup funding trends, investor behavior, deal success rates, and industry-wise performance.

## Dataset Used
Shark Tank India Dataset

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed
- Data Cleaning and Preprocessing
- Total Deals and No Deal Analysis
- Episode-wise Funding Trends
- Industry-wise Pitch Distribution
- Highest Funded Startups
- Shark-wise Number of Deals
- Shark-wise Total Investment Analysis
- Startup Ask Amount vs Deal Amount Comparison
- Success Rate of Pitches
- Top Funded Industries

## Key Insights
- Some sharks invested more frequently than others.
- Consumer-focused businesses received strong attention.
- Several pitches did not convert into deals.
- Higher ask amount did not always guarantee higher funding.
- Certain industries attracted repeated investments.

## Conclusion
This project demonstrates how Exploratory Data Analysis (EDA) can reveal startup investment patterns and business trends using real-world data.
